The Average Game

This is a simulation produced on a daily basis from every killscreen entry in the database. Each killscreen game has been taken and averaged into a single game: "the average game". Every scoring metric of every game has been carefully compiled into this single game: every barrel, pie, spring, rivet, death score, and death location. With each new killscreen entry in the database, the average game automatically adjusts itself.
